<h1 align="center"> canned-laravel-helper </h1>

<p align="center"> laravel 辅助函数.</p>


## Installing

```shell
composer require curatorc/canned-laravel-helper
```

## Usage

### Publish Basic Class

```shell
php artisan vendor:publish --provider="CuratorC\CannedLaravelHelper\ServiceProvider"
```
